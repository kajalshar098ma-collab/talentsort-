
# 💼 Talent Sort

## 🚀 Smart Resume Builder & Direct Hiring Platform

Talent Sort is a modern web-based resume builder and hiring platform where:

- Users can create professional resumes
- Companies can directly hire candidates
- Resume scoring is available
- AI chatbot helps users
- Interview scheduling is included
- Admin panel manages applications
- No third-party involvement

---
